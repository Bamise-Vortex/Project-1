The Structure of HTML

HTML - Hyper Text Markup Language

HTML tag is the root element of HTML document

Head tag are invisible tags for the Front-end development of a website but can be seen at the Back-End development

Title tag reprsents the document title or name

Body tag is used to input the content of a website

Comment in HTML is not a major part because they are not seen on the Front-end of the website but the Back-end. It helps other readers understand what your code
